# Dashboard Detail Spec

## Sidebar
- Top: "Wokku.dev" brand, below: Dokku version + Wokku CE version
- Nav: Dashboard, Apps, 1 Click Deploy, Resources, Activity
- Server menu: admin-only
- Bottom: Documentation, Support, Settings

## Navbar
- Left: search box (functional)
- Right: notification icon (link), terminal icon (link), Deploy button (link to templates), profile badge (dropdown)

## Dashboard Main Content
1. **Stats bar** (4 cards): Total Apps, Active Resources, Errors, Cost MTD
2. **Recently updated apps** (6 cards grid, link to apps page)
3. **Combined activity logs** for all apps
4. **News section**: Wokku/Dokku updates, service news
5. **Random insights** cards

## App cards
- Use terminal/command-line icon
- Show app name, status badge, server, last updated
