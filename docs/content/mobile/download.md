# Download

Get the Wokku mobile app for iOS and Android.

## Download

| Platform | Link |
|----------|------|
| **iOS** | [App Store](https://apps.apple.com/app/wokku) |
| **Android** | [Google Play](https://play.google.com/store/apps/details?id=dev.wokku.app) |

## Features

- View and manage all your apps
- Deploy templates from your phone
- Real-time app status and logs
- Push notifications for deploys and failures
- Restart, stop, and start apps
- Manage environment variables
- View server health
- Team management

## Requirements

- iOS 15+ or Android 10+
- A Wokku account
