# Push Notifications

Get notified on your phone about deploys, failures, and server events.

## Enable Push Notifications

1. Open the Wokku app
2. Go to **Settings → Notifications**
3. Enable push notifications when prompted
4. Select which events you want to receive

## Events

| Event | Description |
|-------|-------------|
| **Deploy success** | App deployed successfully |
| **Deploy failure** | App deploy failed |
| **Server down** | Server became unreachable |
| **Server recovered** | Server came back online |

## Managing Preferences

Tap **Settings → Notifications** in the mobile app to:

- Toggle notifications per event type
- Mute notifications for specific apps
- Set quiet hours
