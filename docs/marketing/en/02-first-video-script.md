# The First Video — Your Introduction Script

> This is the most important video you'll make. Pin it on your profile. Every new follower should see it first. It tells your story in 60-90 seconds.

---

## Title options (A/B test these)

1. **"15 years of coding. Now I'm building products with AI."**
2. **"Why a veteran developer started over in 2026."**
3. **"15-year senior dev, 6-month AI apprentice. This is why."**
4. **"I've been coding since 2010. Here's why I started a TikTok."**

**Recommended: #1** — clearest value prop, highest curiosity hook.

---

## Full Script (90 seconds, screen-only)

**[0-3 seconds] Hook**

*Screen: Your code editor, maybe a terminal showing `git log` with old commits from 2018-2022*

*Voice (confident, matter-of-fact):*
> "I've been a developer for 15 years. Today I'm going to show you why I started over."

**[3-15 seconds] Who you are**

*Screen: Scrolling through an old project codebase or GitHub profile, maybe old code screenshots*

*Voice:*
> "Started coding in 2010. PHP first, then Rails, Node, React, the usual. Most of my career I was a freelancer — teams always changing, no big community, head down in code."

**[15-30 seconds] The shift**

*Screen: Cut to Claude Code in action, prompting and watching code generate*

*Voice:*
> "This year, something changed. I started using Claude Code — an AI agent that writes code with me. And at first, I was skeptical. 15 years of writing code by hand, why would I let an AI do it?"

**[30-50 seconds] The realization**

*Screen: Split or cut — show Wokku dashboard, then Claude Code terminal, then running tests*

*Voice:*
> "But here's what happened. I started building a product by myself. A platform as a service, like Heroku, called Wokku. In two months, solo, with Claude Code, I built something I couldn't have built alone in a year. Not because the AI writes better code than me — it doesn't. But because it handles the boring parts so I can focus on the interesting ones."

**[50-75 seconds] What this channel is**

*Screen: Your editor, terminal, Wokku dashboard — cycle through*

*Voice:*
> "So I'm going to share this. The code, the bugs, the wins, the embarrassing mistakes. How I actually use AI agents. What works, what doesn't, and what I'm still figuring out. In English, because I want to learn with as many developers as possible."

**[75-90 seconds] Ending beat**

*Screen: Your Wokku dashboard, or a clean terminal prompt waiting for input*

*Voice (slower, more personal):*
> "15 years in, I'm genuinely excited about code again. I don't know where this goes. But let's find out together. Follow along."

*End card: Your handle + "Follow for agentic engineering"*

## Production notes

### Screen visuals (what to show)

**Segment 1 (0-15s): Establishing yourself**
- Old code from real projects you've shipped
- Git log showing years of commits
- GitHub contribution graph
- Maybe a screenshot of an old project website (Wayback Machine works)

**Segment 2 (15-30s): The tool**
- Claude Code terminal/UI actively generating code
- Prompt being typed
- Code appearing in editor
- Tests passing

**Segment 3 (30-50s): The product**
- Wokku dashboard
- Deploying an app live
- Build logs streaming
- App going live in browser

**Segment 4 (50-75s): The promise**
- Cycle through: editor, terminal, Claude Code, Wokku
- Shows variety of what you'll be sharing
- Quick cuts (2-3 seconds each) for energy

**Segment 5 (75-90s): The invitation**
- Clean, single shot — maybe Wokku dashboard
- Lets the voice breathe
- Ending calm, intimate

### Voice direction

**Energy levels through the video:**

- **0-3s:** Medium-high (hook)
- **3-15s:** Medium (casual introduction)
- **15-30s:** Building intensity (tension — skepticism)
- **30-50s:** Higher energy (excitement about discovery)
- **50-75s:** Medium (declaring intention)
- **75-90s:** Lower, slower (personal, vulnerable)

**Key moments to nail:**

1. **The hook** — say it like you mean it, not like you're reading. Pause slightly after "15 years of coding."

2. **"I was skeptical"** — let this land. Small pause. This is where you earn trust.

3. **"In two months, solo, with Claude Code"** — emphasize "solo". This is the flex that's also humble.

4. **"15 years in, I'm genuinely excited about code again"** — slow down here. This is the emotional core. Don't rush it.

5. **"Let's find out together"** — warm, inviting. Not a command, an invitation.

### Audio requirements

This video must have **perfect audio** because there's no face to fall back on.

- Use a lavalier mic or USB mic (not phone built-in)
- Record in a quiet room (no AC, no traffic)
- Record the voice separately from the screen if possible (better quality)
- Add subtle background music at 10-15% volume (CapCut has free options)
- Leave small pauses between sentences (don't rush)

### Editing notes

- **Add captions.** Most people watch silent. Captions are non-negotiable.
- **Keep cuts tight.** No dead air. Every second earns its place.
- **Match visuals to voice.** When you say "Claude Code," show Claude Code. When you say "Wokku," show Wokku. Don't let visuals lag behind words.
- **End with a clear CTA.** "Follow for more" or your handle visible.

## Pinning and placement

1. **Post to TikTok first**, wait 24 hours for algorithm bump
2. **Post to Instagram Reels** as second platform
3. **Post to YouTube Shorts** as third
4. **Pin on all three profiles** so new followers see it first
5. **Embed on wokku.dev** — add a section "Who built this?" with this video

---

## Expected performance

**Realistic expectations for first video:**

- TikTok: 100-500 views (algorithm gives new accounts a small test audience)
- Instagram Reels: 50-200 views (algorithm is harder on Reels)
- YouTube Shorts: 20-100 views (hardest platform for new accounts)
- LinkedIn: 200-1,000 views if you have any connections at all

**Do not be discouraged if first video underperforms.** The first video is rarely the viral one. It's the anchor. It needs to exist so when OTHER videos go viral, viewers can click back and understand who you are.

---

## After you post

1. **Don't check metrics every 5 minutes.** Check at 24 hours, 3 days, 7 days.
2. **Reply to every single comment.** Even "nice video". This trains the algorithm and builds community.
3. **Don't delete it if it flops.** Low first-video views are normal. It improves retroactively as you gain followers.
4. **Use it in bio and website.** Link everywhere.
5. **Don't make a "second intro video" for months.** Commit to this one.

---

## One more thing

Before you record, watch 5 successful "intro" videos from developers you respect. Not to copy, but to see what energy works.

Good examples to study:
- Fireship's channel trailer
- ThePrimeagen's "Why I stream" video
- Theo t3.gg's intro video
- Any Marc Lou or levelsio "why I build" video

Notice: they all have clear personality, short runtime, specific promise, and end with invitation.

**Your version of this doesn't need to match them.** It needs to match YOU. The audience that finds your video will either resonate with your vibe or they won't. Either way is fine.

**Record it imperfectly. Post it imperfectly. Iterate later.**

The best first video is the one that exists.
