## Posting Schedule Template

### Optimal posting times for Indonesian audience (WIB/GMT+7)

**TikTok:**
- **Best**: 19:00 - 22:00 (after work, leisure browsing)
- **Second best**: 12:00 - 13:00 (lunch break)
- **Worst**: 02:00 - 06:00

**Instagram Reels:**
- **Best**: 20:00 - 23:00
- **Second best**: 06:30 - 08:30 (morning commute)
- **Worst**: 14:00 - 16:00

**LinkedIn:**
- **Best**: Tuesday/Wednesday/Thursday 08:00 - 09:00
- **Avoid**: Weekends, Monday mornings

**Twitter/X:**
- **Best**: 09:00 - 11:00 and 20:00 - 22:00
- **Weekends**: 09:00 - 11:00

### Weekly cadence (sustainable for solo founder)

| Day | Platform | Content Type |
|-----|----------|-------------|
| Monday 20:00 | TikTok + IG Reels | 60-sec demo (Script series) |
| Tuesday 09:00 | LinkedIn | Text post, longer story |
| Wednesday 20:00 | TikTok + IG Reels | Another demo |
| Thursday 09:00 | Twitter thread | Technical deep-dive |
| Friday 20:00 | TikTok + IG Reels | Face-on-camera content |
| Saturday | OFF | Rest, engage with comments |
| Sunday 10:00 | Twitter | Week recap, numbers |

3 video posts + 1-2 text posts per week = sustainable, not overwhelming.

---

## Content Cross-Posting Rules

1. **TikTok first**, then Instagram Reels 24 hours later (TikTok algorithm penalizes reposts)
2. **LinkedIn and Twitter use different copy** — never cross-post verbatim
3. **All platforms link to wokku.dev** — never to social bio that redirects
4. **Shorten links with umami or bit.ly** to track which platform converts
