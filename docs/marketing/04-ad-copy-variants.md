## Creative Iteration Rules

1. **Test 3-5 variants per platform simultaneously**
2. **Kill ads with CTR < 1% after 1,000 impressions**
3. **Scale ads with CTR > 2% and signup rate > 10%**
4. **Refresh creatives every 2 weeks** (ad fatigue is real)
5. **Test different hooks**, same body copy
6. **Test different images/videos**, same hook
7. **Never test multiple variables at once** — you won't know what changed

## Tracking Setup

**Required before running ads:**

1. **UTM parameters on every link:**
   ```
   wokku.dev/?utm_source=facebook&utm_medium=cpc&utm_campaign=qris-hook&utm_content=variant-1
   ```

2. **Conversion tracking:**
   - Meta Pixel installed
   - Google Analytics (or Plausible — use your own template)
   - Track signups, paid conversions, and lifetime value

3. **Attribution model:**
   - First touch: which ad brought them in
   - Last touch: which ad closed the sale
   - For most SaaS, last-touch + 7-day window works

4. **Cohort analysis:**
   - Group users by ad source
   - Compare retention at day 7, 30, 90
   - Kill sources with low retention even if conversion is high
